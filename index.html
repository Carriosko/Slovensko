League of Legends Stagecast Quiz Script


How to install

1. Install Resource Override Extension

2. Download ROrules.json

- Right Click

- Save As

3. Open Resurce Override Menu

- Right Click

- Copy Link Address

- Open new tab in browser

- Paste copied link and press Enter

4. Click on Options, in the top right corner, then on Load Rules

5. Select the just downloaded ROrules.json file

6. Refresh the Quiz Page

7. Play the game. If you lose script will send your score to server - score is as high as possible

ROrules file is updated on a daily basis, since the quiz change/ish every 24h

So remember to check if a new version is available and if it is, repeat all steps from step 2
